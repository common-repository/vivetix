== Description ==

This plugin adds a shortcode that load an event's widget from vivetix.

It automatically resizes the content from the vivetix, to fit in the site.

== Frequently Asked Questions ==

= Why does this plugin exist? =

This plugin exists to facilitate the insertion of the vivetix event's widget on wordpress sites.

= How to use this plugin? =

1. Enter to admin area
2. On the top, click add plugin and install it
2. Activate the plugin.
3. Create a new post/page.
4. Add a new block called shortcode and add this code: [vivetix url=YOUR-VIVETIX-URL]
